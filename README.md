Government Scheme Allocation & Efficiency Analysis
📌 Project Overview

This project analyzes Indian government schemes across multiple sectors using Python for Exploratory Data Analysis (EDA) and 
Tableau for dashboard visualization.
The analysis focuses on:
budget allocation
scheme efficiency
beneficiary coverage
utilization trends
state-wise performance

🎯 Objectives
Analyze budget allocation trends
Compare sector-wise efficiency
Identify inefficient schemes
Study state-wise performance
Build an interactive Tableau dashboard

💡 Key Insights
Higher budget allocation does not always indicate better efficiency.
State-wise efficiency differences exist across sectors.
Some low-budget schemes show strong impact and utilization.
